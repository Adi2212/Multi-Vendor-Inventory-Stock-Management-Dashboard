# Multi-Vendor-Inventory-Stock-Management-Dashboard

A modern, multi-tenant inventory and stock management dashboard.

## Features
- Super Admin and Vendor Admin Roles
- Product and Stock Management
- Supplier and Purchase Tracking (Restock)
- Point of Sale (POS) and Invoicing
- Dashboard Analytics
- JWT Authentication

## Stack
- Frontend: React, Tailwind CSS, Vite
- Backend: Java, Spring Boot, MySQL
